[expression for item in iterable if condition]

expression → what you want to output
item → each element from the iterable
iterable → list, tuple, string, range, etc.
condition (optional) → filter elements

1. Simple list creation

nums = [1, 2, 3, 4]
squares = [x**2 for x in nums]
# [1, 4, 9, 16]

2. With condition

evens = [x for x in nums if x % 2 == 0]
# [2, 4]

3. Inline condition (if-else)

result = ["even" if x % 2 == 0 else "odd" for x in nums]
# ['odd', 'even', 'odd', 'even']
